## Informasi

Ini adalah panduan tentang cara membuat situs statis dengan IsmyBio. Ini adalah cara yang bagus untuk memamerkan proyek, resume, atau informasi lain yang ingin Anda bagikan dengan dunia. Pastikan untuk mengikuti langkah-langkah di bawah ini untuk membuat situs statis Anda.

## Langkah-langkah

<Steps>
<Step>

### Masuk ke Akun Anda di IsmyBio

Klik di sini untuk [Login](https://ismy.bio/login)

</Step>
<Step>

### Navigasikan ke Halaman Situs Statis

Klik menu Situs Statis di sisi kiri dasbor.

</Step>
<Step>

### Upload File Situs

Klik tombol `Upload` untuk mengunggah berkas Anda. Anda dapat mengunggah berkas HTML, CSS, JS, Gambar, dan Zip.

</Step>
<Step>

### Periksa Hasil

Setelah Anda mengunggah file Anda, Anda dapat melihat situs statis Anda dengan mengunjungi URL.

</Step>
</Steps>

## Struktu File & Folder

Jika Anda mengunggah file Zip, pastikan bahwa Struktur Folder dan File seperti yang ditampilkan di bawah ini:


<Files>
  <Folder name="assets" defaultOpen>
  <Folder name="css" defaultOpen>
  <File name="style.css" />
  </Folder>
  <Folder name="img" defaultOpen>
  <File name="image.png" /> 
  </Folder>
  <Folder name="js" defaultOpen>
  <File name="example.js" />
  </Folder>
  </Folder>
  <File name="index.html" />
</Files>


<Callout type="info">
Pastikan konten file zip Anda memiliki file index.html saat diekstrak. Pastikan Anda merujuk gambar, js, CSS...dll dalam format relatif berikut.
<strong>Ukuran File Maksimum untuk diunggah adalah 10MB.</strong>
</Callout>
