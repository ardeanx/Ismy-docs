## Information

This is a guide on how to create a static site with IsmyBio. This is a great way to showcase your projects, resume, or any other information you want to share with the world. Make sure to follow the steps below to create your static site.

## Steps

<Steps>
<Step>

### Log in to your Account at IsmyBio

Click here to [Login](https://ismy.bio/login)

</Step>
<Step>

### Navigate to the Static Site Page

Click on the Static Site menu on the left side of the dashboard.

</Step>
<Step>

### Upload your Files

Click on the `Upload` button to upload your files. You can upload HTML, CSS, JS, Image, and Zip files.

</Step>
<Step>

### View your Static Site

After you upload your files, you can view your static site by visiting the URL.

</Step>
</Steps>

##  File & Folder Structure

If you upload a Zip file, make sure that the Folder and File Structure is like displayed below:


<Files>
  <Folder name="assets" defaultOpen>
  <Folder name="css" defaultOpen>
  <File name="style.css" />
  </Folder>
  <Folder name="img" defaultOpen>
  <File name="image.png" /> 
  </Folder>
  <Folder name="js" defaultOpen>
  <File name="example.js" />
  </Folder>
  </Folder>
  <File name="index.html" />
</Files>


<Callout type="info">
Make sure your zip file contents have an index.html file when unzipped. Make sure you reference your images, js, css...etc in the following relative format.
<strong>Max File Size for upload is 10MB.</strong>
</Callout>
