## Informasi

Halaman splash merupakan bagian pembuka sebelum Anda memasukkan tautan. Halaman splash dapat meningkatkan profesionalisme dan kehadiran merek Anda kepada audiens Anda. Pastikan untuk mengikuti langkah-langkah berikut:

![Splash](/splash.jpeg)

## Langkah-langkah

<Steps>
<Step>

### Masuk ke Akun IsmyBio Anda

Klik disini untuk [Login](https://ismy.bio/login)

</Step>
<Step>

### Navigasikan ke Halaman Pembuka

Klik menu Halaman Pembuka pada sisi kiri dasbor

</Step>
<Step>

### Buat Halaman Pembuka

Klik tombol `Tambah Baru` untuk memulai.

</Step>
<Step>

### Penyesuaian dan Kustomisasi

Sesuaikan dan Kustomisasi sesuai dengan keinginan

</Step>
</Steps>

## Properti Halaman Pembuka

Anda dapat menyesuaikan beberapa atribut ini pada Halaman Pembuka:

- Logo
- Favicon (Web Icon)
- Gambar OpenGraph
- Penghitung Mundur (Sebelum Menampilkan Konten)
- Pengalihan Otomatis
- Tombol URL dan Nama Kedua
- Iklan (Header & Footer)
- CSS & JS Khusus
