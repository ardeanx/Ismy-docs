## Informasi

Di halaman ini Anda akan memahami cara membuat file tautan di IsmyBio. Pastikan untuk mengikuti langkah-langkah di bawah ini untuk membuat file tautan.

![Images](/filelinks.jpeg)

## Langkah-langkah

<Steps>
<Step>

### Masuk ke Akun Anda di IsmyBio

Klik disini untuk [Login](https://ismy.bio/login)

</Step>
<Step>

### Navigasi ke Halaman Link File

Klik menu `Link File` di sisi kiri dashboard.

</Step>
<Step>

### Buat Link File

Klik tombol `Upload File` untuk membuat link file baru.

</Step>
<Step>

### Upload File

Unggah file yang ingin Anda bagikan dan sesuaikan URL nya.

</Step>
</Steps>

## Tujuan

Fungsi utama dan Tujuan Fitur ini:

- Untuk menyimpan File Anda secara online
- Untuk berbagi file Anda dengan orang lain tanpa berbagi langsung
- Untuk mengunggah konten Anda dan membagikannya secara global dengan link
