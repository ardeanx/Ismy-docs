## Information

In this page you will understand how to create a link file in IsmyBio. Make sure to follow the steps below to create a link file.

![Images](/filelinks.jpeg)

## Steps

<Steps>
<Step>

### Log in to your Account at IsmyBio

Click here to [Login](https://ismy.bio/login)

</Step>

<Step>

### Navigate to the Link File Page

Click on the `File Links` menu on the left side of the dashboard.

</Step>

<Step>

### Create a New Link File

Click on the `Add New` button to create a new link file.

</Step>

<Step>

### Upload the File

Upload the file you want to share and adjust the Short URL.

</Step>

</Steps>
