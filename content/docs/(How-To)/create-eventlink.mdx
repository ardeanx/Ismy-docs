## Information

At IsmyBio you can create an Event Links to integrate it with the device calendar system and much more. Make sure to follow these steps:

![Event](/event.jpeg)

## Steps

<Steps>

<Step>

### Log in to your Account at IsmyBio

Click here to [Login](https://ismy.bio/login)

</Step>
<Step>

### Navigate to the Event Links Page

Click on the Event Link menu on the left side of the dashboard.

</Step>
<Step>

### Create Event Link

Click on the `Event Link` button to start.

</Step>
<Step>

### Adjusting & Customize

Type the Event Links URL, after that adjust the information.

</Step>
</Steps>
