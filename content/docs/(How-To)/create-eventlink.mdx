## Informasi

Di IsmyBio Anda dapat membuat Tautan Acara untuk mengintegrasikannya dengan sistem kalender perangkat dan banyak lagi. Pastikan untuk mengikuti langkah-langkah berikut:

![Event](/event.jpeg)

## Langkah-langkah

<Steps>
<Step>

### Masuk ke Akun Anda di IsmyBio

Klik disini untuk [Login](https://ismy.bio/login)

</Step>
<Step>

### Navigasi ke Halaman Link Acara

Klik menu Link Acara di sisi kiri dasbor.

</Step>
<Step>

### Buat Link Acara

Klik tombol `Tambah Baru` untuk memulai.

</Step>
<Step>

### Penyesuaian dan Kustomisasi

Ketik URL Tautan Acara, setelah itu sesuaikan informasinya.

</Step>
</Steps>
