## Langkah-langkah

<Steps>
<Step>

Klik disini untuk [Login](https://ismy.bio/login)

</Step>

Klik menu Pixel di sisi kiri dasbor.

</Step>
<Step>

## Tujuan Pixel Pelacakan

Anda mungkin bertanya-tanya apa tujuan atau manfaat menggunakan Pelacakan Piksel. Berikut alasannya:

- Pelacakan Kinerja
- Penargetan Ulang & Periklanan Perilaku
- Pelacakan Konversi

### Bagaimanca cara Pixel bekerja?

1. URL piksel pelacakan dibuat oleh platform analitik atau iklan.
2. Gambar piksel (biasanya GIF atau PNG transparan 1x1) ditempatkan di halaman web, iklan, atau email.
3. Saat pengguna memuat halaman atau membuka email, browser meminta gambar dari server.
4. Server mencatat aktivitas pengguna, yang kemudian dianalisis untuk mendapatkan wawasan pemasaran.
